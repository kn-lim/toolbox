# toolbox

[![release](https://github.com/kn-lim/toolbox/actions/workflows/release.yaml/badge.svg?branch=main)](https://github.com/kn-lim/toolbox/actions/workflows/release.yaml)

This repo is used as the image my [homelab](https://github.com/kn-lim/homelab) uses in Argo Workflows.

To create the image, [mise](https://github.com/jdx/mise) is used to install all the tools I'll need, while [Renovate](https://github.com/apps/renovate) manages the tool versions.
